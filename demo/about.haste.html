About this demo!

The page uses a sidebar template layout which itself includes the base layout showing how you can "compose" templates together in an extending manner.

Below you can find three counters. Each of these are the same included file but they are provided variables differently to set the starting count value.

The first takes it's value from the template-level variable defined at the top of the file.
The second is provided a starting point via an attribute.
The third is provided a starting point using a variable tag.

This shows how attributes will override template-level variables.
